# 🎯 老板人格测评系统

**测测你的老板是什么类型，学会如何与他相处**

## 📖 项目简介

这是一个诙谐幽默、带有讽刺风格的老板人格测评系统。通过 30 道情境选择题，基于 8 个维度将你老板的人格类型匹配到 48 种不同的老板原型中。

每个类型包含：
- ✨ 类型名称（如"考古学家型"、"时间旅行者型"）
- 💬 典型 Slogan（真实还原老板口头禅）
- 📊 8 个维度得分雷达图
- 📖 详细类型解析
- 🎭 行为特征描述
- 💬 沟通技巧建议
- 🛡️ 应对策略指南

## 🚀 快速开始

### 方法一：直接打开（推荐）
直接双击 `index.html` 文件即可在浏览器中打开使用。

### 方法二：本地服务器
```bash
# Python 3
python -m http.server 8080

# 访问 http://localhost:8080
```

### 方法三：使用 Live Server
如果你使用 VS Code，安装 Live Server 插件后右键 index.html 选择"Open with Live Server"。

## 🎮 使用说明

1. **开始测评**：点击"开始测评"按钮
2. **回答问题**：共 30 道题，每题 4 个选项
3. **查看结果**：完成后生成详细报告，包括：
   - 老板类型名称和 Slogan
   - 8 个维度得分雷达图
   - 类型详细解析
   - 沟通技巧和应对策略
4. **重新测评**：可多次测评取平均值，或分享给朋友一起测

## 💡 使用场景

- 😂 **个人娱乐**：吐槽老板，缓解工作压力
- 🎯 **职场生存**：了解老板类型，提升沟通效率
- 👥 **团队分享**：增进同事间对管理的理解
- 📚 **管理学习**：反向学习，避免成为这样的老板

## ⚠️ 温馨提示

- 本测评仅供娱乐，请勿过于认真
- 结果不代表专业心理评估
- 如有雷同，纯属巧合
- 建议匿名使用，保护好自己
