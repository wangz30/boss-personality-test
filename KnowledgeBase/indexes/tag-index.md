## 📝 标签使用规范

### 命名规则

- 使用小写字母
- 单词间用连字符 `-`
- 保持简洁明了
- 优先使用现有标签

### 示例

```markdown
#正确
#priority-high
#status-implemented
#authentication
#cqrs-pattern

#错误
#PriorityHigh          # 不应使用大写
#status implemented    # 不应有空格
#超重要的优先级        # 应使用英文
```

### 标签层次

```
#领域/子领域/具体
#tech/database/postgresql
#feature/auth/authentication
```
