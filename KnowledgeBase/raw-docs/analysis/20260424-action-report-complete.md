#### 优化项3：修复boss-types.json数据不一致

**问题**：48个类型的`dimensions`(1-6)与`expectedScores`(0-33)不匹配

**修复前示例**：
```json
{
  "dimensions": {"communication": 3, ...},
  "expectedScores": {"communication": 12, ...} // 不匹配
}
```

**修复逻辑**：`expectedScores = (dimensions - 1) * 3`

**修复后**：
```json
{
  "dimensions": {"communication": 3, ...},
  "expectedScores": {"communication": 6, ...} // 3-1=2, 2*3=6 ✓
}
```

**验证**：运行验证脚本，48类型100%通过
